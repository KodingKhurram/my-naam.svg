<h2> Overview </h2>
<span> <strong>my-naam.svg</strong> is a simple svg generator with couple of themes and fonts, simple to use with CSS namespaces.</span>

<h2> Parameters </h2>
 <details>
  <summary>global parameters</summary>
  <ul>
    <li>height: - specify height of svg.</li>
    <li>width: - specify width of svg. </li>
  </ul>
</details>
 <details>
  <summary>theme</summary>

  Theme is the root paramater of svg generator, there are currently 4 themes namely,
  <ol>
    <li>flicker</li>
    <li>glow</li>
    <li>tranform</li>
    <li>wave</li>
  </ol>

  ```
  ?theme=flicker
  ```

</details>
<details>
  <summary>text</summary>
  The text for svg to contain.

  ```
  ?theme=flicker&text1=my-&text2=naam&text3=.svg
  ```
  ```
  ?theme=glow&text=my-naam.svg
  ```
  ```
  ?theme=transform&text=my_naam.svg
  ```
  ```
  ?theme=wave&text=my-naam.svg
  ```

</details>
<details>
<summary>CSS utilities</summary>
  <ul>
    <li>background-color</li>
    <li>font</li>
    <li>font-size</li>
    <li>font-weight</li>
    <li>letter-spacing</li>
    <li>color</li>
    <li>text-align</li>
  </ul>
</details>
<details>
  <summary>special parameters</summary>
  <ul>
    <li>container: - Only applicable on <strong>flicker</strong> theme, default value "on".</li>
    <li>border-color: - Only applicable on <strong>flicker</strong> theme, default value "#FF00E6". </li>
    <li>transform-color: - Only applicable on <strong>transform</strong> theme, default value "#37b9f1".</li>
  </ul>
</details>

More detailed documentation in demo page :)
